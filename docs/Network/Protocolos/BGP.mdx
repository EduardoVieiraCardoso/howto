# Introdução ao protocolo de Roteamento BGP

Existem alguns protocos de roteamento no mercado, porem para roteamento de borda (Internet), o mundo todo utiliza o protocolo BGP, que em tradução livre, 
significa "Protocolo de Roteamento Borda".

Geralmente este protocolo é utilizado por provedores de internet (ISPs, Internet Service Providers)
